# Azure ETL Pipeline with Power BI Dashboard

## 📌 Project Overview
This project demonstrates an end-to-end data engineering pipeline using Azure Data Factory and Power BI.

## 🛠 Tools Used
- Azure Data Factory
- Azure Blob Storage
- Power BI

## 📊 Dataset
Online Retail Dataset containing sales transactions.

## 🔄 Pipeline Steps
1. Data ingestion from source
2. Data transformation (TotalAmount, YearMonth)
3. Data stored in Azure Blob (CSV)

## 📈 Power BI Dashboard
- Total Revenue KPI
- Monthly Sales Trend
- Top 10 Products
- Revenue by Country
- Top Customers

## 🔍 Key Insights
- Top products contribute majority of revenue
- Sales show seasonal trends
- Certain countries generate higher revenue
